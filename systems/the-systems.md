# The Systems

A menu, not a checklist. Steal what fits this season of your life and ignore the rest. Each is meant to be shrunk until it is easy, then run on the noted rhythm. Add **one at a time** (see [Start With One System](start-with-one.md)).

> Inspired by Ben Meer's *Year of Systems*, reorganized around my own priorities — with Relationships moved to the front, because it is the area that quietly compounds the most and gets neglected the most.

## Relationships (start here)
| System | What it is | Rhythm |
|---|---|---|
| **Keep in Touch** | The people you love, each on a cadence; a tool tells you who's overdue. [Full system →](keep-in-touch-system.md) | weekly |
| Standing dates | One recurring thing on the calendar per close relationship (call, walk, game night) so there's nothing to initiate. | monthly |
| Do-a-thing-together | A shared puzzle / word game / show you compare notes on. Warmth without small talk. | ongoing |
| Birthday heads-up | A single list with a few days' warning, so you're never scrambling. | quarterly to set |
| The forward file | A running note of who loves what, so "saw this, thought of you" writes itself. | ongoing |
| Inner circle check | Once a season, ask: who am I drifting from that I don't want to? Adjust cadences. | quarterly |

## Mind
| System | What it is | Rhythm |
|---|---|---|
| Reading capture | One place every article/book you want lands, so it's not fifty open tabs. | ongoing |
| Decision journal | Write the decision and why, before the outcome. Learn from your actual reasoning. | as needed |
| Information diet | Decide once what you let in (and mute), instead of re-deciding hourly. | set once |
| Weekly reset block | 20 quiet minutes to look back and look ahead. Not a productivity ritual — a breath. | weekly |

## Physical health
| System | What it is | Rhythm |
|---|---|---|
| Evening wind-down | One small repeatable cue that ends the day (same three steps). | daily |
| Minimum viable workout | The smallest movement you'll actually do on a bad day. Consistency over intensity. | daily |
| Meal plan | Decide the week's dinners once; stop the daily 5pm "what's for dinner" tax. | weekly |
| Appointments on autopilot | Book the annual stuff a year ahead the day you leave the office. | annually |

## Money
| System | What it is | Rhythm |
|---|---|---|
| Autopay everything safe | Bills you can safely automate, automated once. | set once |
| Automated savings | A small transfer that happens without you. | set once |
| Subscription sweep | Twice a year, cancel what you forgot you pay for. | semiannually |
| Important docs vault | One secure place for the documents future-you will panic looking for. | annually |

## Environment
| System | What it is | Rhythm |
|---|---|---|
| Everything has a home | Decide where things live once; tidying becomes "return," not "decide." | set once |
| Weekly desk/inbox reset | One pass, one day, not a constant background guilt. | weekly |
| Capsule basics | Fewer, better defaults so daily choices shrink. | seasonal |
| The reset day | A quarterly "adulting day" for the errands that pile up. | quarterly |

---
This is a menu of starting points, not a syllabus. Pick the one that would make *this* week lighter and set it up today. When it's invisible, come back for the next.
